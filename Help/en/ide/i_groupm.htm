What's New in the MIDletPascal 3.x IDE \ Group Management



since 3.0

Now you can work with many projects at the same time with the new Group Management feature. This is very useful if you have large projects with many different versions of the same MIDlet (may be different language versions, may be different editions, etc).

You can create a new Project Group via the File \ New \ Group menu item, and switch the Group Manager visibility via the View \ Group Manager menu item.
