What's New in the MIDletPascal 3.x IDE \ Integrated Editors


Hexadecimal Editor
since 3.0

Now any binary file (not been an image) open or dropped into the IDE, will be open with the internal hexadecimal editor.

 Customizable Highlighting
since 3.0

In Settings \ Code Editor Style you will find all the available styles for the code editor.

 Code Expanding
since 3.0

This feature is like the one in Delphi, just type the code abbreviations you desire to expand and press Ctrl+J.

Here is a list of the available abbreviations you can expand into code:

    arrayd - array declaration (var)
    arrayc - array declaration (const)
    fors - for (without begin/end)
    forb - for (with begin/end)
    function - function declaration
    ifs - if (without begin/end)
    ifb - if (with begin/end)
    ife - if then (without begin/end) else (without begin/end)
    ifeb - if then else
    procedure - procedure declaration
    whileb - while (with begin/end)
    whiles - while (without begin/end)

 Open File at Cursor
since 3.0

This feature is like the one in Delphi, just press Ctrl+ENTER on a unit in the uses clause and it will be open.

 Help On Code
since 3.0

This feature is like the one in Delphi, just press Ctrl+F1 on a compiler 'magic' procedure or function call and it will be open its related help topic.

 Class File Browser
since 3.5

This feature allows you to browse the generated class files for your project. Just click in the ".class" tab in the code editor.
