What's New in the MIDletPascal 3.x IDE \ GUI Items


Drag & Drop Support
since 3.0

Now you can drop any kind of file to the IDE.

 Recents Project Menu
since 3.0

In the File \ Reopen submenu you will find the most recently used projects so you can quickly reopen them.

 Full Screen Switch
since 3.0

In the View \ Full Screen menu item you have a quick switch to a full screen visualization mode. Invokable also via the F11 key.

 Internationalization
since 3.0

In the View \ Language submenu you can find the available translations for the IDE strings.

 Welcome Page
since 3.3

Now you can quickly reopen the most recent projects with just one click. Use the View \ Welcome Page menu item to toggle the Welcome Page visibility.

 Side Bar
since 3.4

Now you can quickly access the most common functions (New, Open, Build, Run and Build & Run) with just one click. Use the View \ Side Bar menu item to toggle the Side Bar visibility.
