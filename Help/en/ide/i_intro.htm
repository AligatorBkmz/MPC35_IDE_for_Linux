What's New in the MIDletPascal 3.x IDE \ Introduction


As an approach to address more-than-hobbyst-concerns, Javier Santo Domingo (j-a-s-d) started coding the 3.0 IDE in the end of October 2009, and set the following goals for the 3.0 version:

  • robust and stable IDE
  • project group management
  • automated building enabled environment (cli tools)
  • full support for conditional defines and source code preprocessing
  • hexadecimal editing for resources
  • multiple language support
  • WINE compatible (for usage in Linux and MacOS)
  • portable (for usage in pen-drives and other devices)
amongst many others which are common requirements in the professional and corporative usage of a development tool.

The main development tool of choice was Turbo Delphi 2006 since its a pascal variant that the community knows very well, its free and provides access to a wide range of third party tools.

For the CLI tools the selected was the Free Pascal compiler (version 2.4.0).

As extra goals were set:

  • some compiler (c-language) tasks already covered in the compiler's Introduction
  • remove AV false alarms mainly occurring with the FW.class stub (java-language)

The brand new 3.0 IDE was built from scratch and does not contain any line of the old one (was not pascal written and also contained usage of commercial third party controls).
