What's New in the MIDletPascal 3.x IDE \ CLI Tools


The Command-Line Interface tools are provided mainly to enable MIDletPascal to be accesible in automated builds scenarios.
Here is described also the IDE command-line handling.

Group Manager
since 3.0

  • Usage:
      mp3clgm <command> <group_fullpath> [<project_fullpath>]
  • Commands:
      group related
      • new
      • remove
      projects related
      • include
      • exclude

    •  Project Manager
    since 3.0

  • Usage:
      mp3clpm <command> <project_fullpath> [<parameter1>] [<parameter2>]
  • Commands:
      project commands
      • new
      • remove
      project midlet info commands
      • setversion
      • setvendor
      • seticon
      project source files commands
      • addsf
      • delsf
      • rensf
      project resource files commands
      • addrf
      • delrf
      • renrf
      • setrfc
      project build configurations commands
      • addbc
      • delbc
      • renbc
      • setabc

    •  IDE Command-Line Handling
    since 3.0

  • Usage:
      mp3ide [<command>] <project_or_group_fullpath>
  • Commands:
      project and group commands
      • build
